Krawings Task Manager

Department-scoped daily task lists for the Krawings Portal PWA.

  • One template per (department, day-of-week) combination.
  • Tasks grouped by day-part: opening, mid-day, closing.
  • Daily instances spawned by cron at 00:05; idempotent.
  • Anyone in the department can complete any task; per-line accountability via hr.employee.
  • Manager can add ad-hoc one-off tasks to today's instance without modifying the template.
