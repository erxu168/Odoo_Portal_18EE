# Brand Assets

Drop the logo files here so the whole team has one place to find them.

**Expected files:**

| File | Purpose |
|------|---------|
| `Final Logo in Yellow and Red.svg` | Web, digital ads, social — scalable master |
| `Final Logo in Yellow and Red.pdf` | Print at any size — signage, menus, flyers |
| `FIXED OUTLINES VERSION.pdf` | Black-and-white / single-color version — stamps, receipts, newsprint |

**To fill this folder** (one-time, ~30 seconds):
```bash
# from wherever you saved the designer's files (e.g. ~/Downloads)
cp ~/Downloads/Final\ Logo\ in\ Yellow\ and\ Red.svg \
   /Users/ethan/Odoo_Portal_18EE/.agents/what-a-jerk/brand/assets/logo-yellow-on-red.svg

cp ~/Downloads/Final\ Logo\ in\ Yellow\ and\ Red\ PDF.pdf \
   /Users/ethan/Odoo_Portal_18EE/.agents/what-a-jerk/brand/assets/logo-yellow-on-red.pdf

cp ~/Downloads/FIXED\ OUTLINES\ VERSION\ copy.pdf \
   /Users/ethan/Odoo_Portal_18EE/.agents/what-a-jerk/brand/assets/logo-black-outline.pdf
```

See [../brand-guide.md](../brand-guide.md) for how to use each variant.
