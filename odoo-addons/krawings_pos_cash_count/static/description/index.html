Krawings POS Cash Count

Count coins by value or quantity — no on-screen keyboard

Improves the cash-counting step of the POS register closing:

  • Each denomination flagged as a coin can be counted by total value (e.g. type "4.50" of 5-cent coins) or by quantity. Notes/bills stay quantity-only.
  • Entry uses the POS' own on-screen numpad popup, so the native (Android) soft keyboard never pops up on tablet tills.
